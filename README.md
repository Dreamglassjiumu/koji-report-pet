# Koji Report Pet

Koji Report Pet（中文名：Koji 日报桌宠助手）是一个纯本地的文案组日报桌宠助手。它面向文案组成员的日常工作记录场景：白天随手记录“今天做了什么”，晚上点击按钮后，由本地模板和关键词规则整理生成中文日报。

Koji 是文案组吉祥物，也是日报小秘书、日报催收员、摸鱼观察员和项目记录小助手。v0.2.1 继续使用 emoji / CSS 占位桌宠，不需要任何正式图片素材，也不会连接任何在线服务。

## 本地预览方式

进入项目目录后运行：

```bash
python -m http.server 48763
```

然后打开：

```text
http://localhost:48763
```

注意：因为用户本地已有很多项目，为避免端口冲突，本项目默认推荐使用 `48763`。不要默认推荐 `3000`、`5173`、`8000`、`8080`、`5000` 等常见端口。

也可以直接打开 `index.html` 进行基础预览，但更推荐使用上面的 Python 本地服务器方式。

## 隐私与技术原则

本项目 v0.2.1 仍然是纯静态网页：

- 不联网
- 不接 AI
- 不调用任何 AI API
- 不上传数据
- 不使用后端服务
- 不使用 npm、构建工具或外部 CDN
- 所有事项、日报和设置都保存在当前浏览器的 `localStorage`

如果更换浏览器、清理浏览器数据或使用无痕模式，已保存内容可能无法继续保留。

## v0.2 新增功能

- Koji 桌宠强化：新增状态气泡、状态标签、快捷按钮、拖动和收起 / 展开。
- 动作测试面板：设置区可一键触发 Koji 12 个状态，方便后续替换动作素材时测试。
- 更好的日报生成：保留正式、简洁、详细、文案组、问题反馈型 5 种模板，并按分类生成更自然的中文总结。
- 复制给 GPT 润色：只生成本地 prompt 并复制到剪贴板，不接入任何 AI API。
- TXT / Markdown 导出：使用浏览器 Blob + `URL.createObjectURL` 本地下载当前日报。
- 自定义日报格式：支持标题 / 称呼、日期、明日计划、问题反馈、输出偏好、常用结尾语。
- 今日简报：生成 3 到 5 条短 bullet，适合快速回顾。
- 周报素材：读取近 7 天 `localStorage` 记录，按分类整理为可复制素材。
- Toast 提示：添加、保存、复制、导出、删除、错误等操作统一非阻塞提示。
- 历史记录优化：日期列表显示星期、事项数量和日报状态，详情区支持复制、导出和删除当日数据。

## Koji 素材替换说明

后续可以把正式素材放入：

```text
assets/koji/idle.png
assets/koji/wave.png
assets/koji/record_ready.png
assets/koji/success.png
assets/koji/thinking.png
assets/koji/writing.png
assets/koji/happy.png
assets/koji/confused.png
assets/koji/angry.png
assets/koji/sleep.png
assets/koji/drag.png
assets/koji/error.png
```

然后在 `app.js` 的 `petStates` 配置里，把对应状态的 `image` 设置为素材路径即可，例如：

```js
idle: {
  key: "idle",
  label: "待机",
  emoji: "🐾",
  image: "assets/koji/idle.png",
  message: "来了，今天干啥了？",
  cssClass: "pet-idle",
  duration: 0,
  soundHint: ""
}
```

显示逻辑为：如果 `image` 存在且加载成功，则优先显示图片；如果没有图片或加载失败，则自动回落到 emoji / CSS 占位，不会出现破图。建议先使用设置区的“Koji 动作测试面板”逐个触发 12 个状态，确认素材差分效果。

## 当前功能说明

- 今日事项记录：支持多行输入、自动记录日期和时间、选择项目标签、本地保存。
- 事项管理：支持编辑、删除，刷新页面后仍可保留。
- 项目标签：内置默认标签，并支持在设置中新增自定义标签。
- 日报生成：支持正式日报、简洁日报、详细日报、文案组日报、问题反馈型日报 5 种模板。
- 本地分类规则：通过关键词将事项粗略归类为资料整理、功能优化、测试验证、沟通协作、文案创作、问题修复和其他工作。
- 日报编辑、保存、复制、GPT 润色 prompt 复制、TXT / Markdown 导出。
- 今日简报与近 7 日周报素材整理。
- 历史记录：按日期倒序显示历史事项和日报，支持复制历史日报、导出历史日报、删除某一天全部数据。
- 设置：支持用户名、默认日报模板、自定义标签、自定义日报格式、Koji 动画开关、清空全部本地数据。
- Koji 桌宠：固定显示在右下角，可拖动，可收起 / 展开，支持 12 个状态：待机、打招呼、准备记录、记录成功、思考、写日报、开心、疑惑、催日报、困倦、被拖动、报错。
