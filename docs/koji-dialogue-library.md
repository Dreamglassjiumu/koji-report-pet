# Koji 台词源文档

> 本文件为台词源文档，程序实际读取 `data/koji-dialogues.js`。

## 语气模式

- `standard` / 标准 Koji：默认模式，欠但不过分。
- `sassy` / 更贱一点：更多吐槽、阴阳怪气与低频安全抽象梗。
- `formal` / 更正经一点：偏工具提示，减少玩笑。
- `mixed` / 中英日混合：中文夹 English 和简单日语。
- `quiet` / 少说话模式：短句、低打扰。

## 状态台词覆盖

程序配置覆盖 13 个状态：`idle`、`wave`、`record_ready`、`collect`、`success`、`thinking`、`writing`、`happy`、`confused`、`angry`、`sleep`、`drag`、`error`。每个语气模式下每个状态至少 5 条短台词。

## 整点报时

整点报时只显示 Koji 气泡，不播放真实声音，不使用系统通知。当前配置重点覆盖 `00`、`01`、`02`、`08`、`09`、`12`、`15`、`18`、`22`、`23` 等容易需要提醒的时段；未覆盖时段由程序回退到通用报时文案。

## 抽象梗 / homo 梗风味安全池

安全池只收录不露骨、不攻击现实群体、不含歧视表达的轻度抽象工作梗，例如“这波是日志经济学”“日报素材 +1”等。程序只在 `sassy` 模式下，且仅对 `collect`、`happy`、`angry`、`writing`、`wave` 等状态低概率混入。

## 英文 / 中英混合备用池

`mixed` 模式使用 `Ready`、`log`、`report`、`save`、`work mode` 等简单英文工作词，避免长句撑破气泡。

## 日语 / 中日混合备用池

`mixed` 模式使用 `おはよう`、`ねむい`、`待って`、`よし`、`やばい`、`desu` 等简单日语语气词，不使用复杂长句。

## 禁用方向

- 不输出露骨性表达。
- 不攻击现实群体。
- 不使用歧视性词汇。
- 不让 Koji 恶毒辱骂用户。
- Koji 可以欠，但不能恶毒。
